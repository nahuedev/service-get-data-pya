# Servicio Get-Data-PYA

RUN : 
```sh
npm i
```
```sh
npm run start
```

## Environments 
crear un .env en la raiz del proyecto

```env
PORT=3000
LATITUD="-34.9661224"
LONGITUD="-54.9445817"
RESULTADOS_MAXIMOS=150
A_PARTIR_DE_CUAL_TRAER=0
```

## EndPoint : 
```js
   api/get-data
```


Nota: Aplicacion con fines de educativos y de aprendisaje.
